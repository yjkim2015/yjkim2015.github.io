###### HTTP (HyperText Transfer Protocol)

HTTP는 W3 상에서 정보를 주고받을 수 있는 프로토콜이다. 주로 HTML 문서를 주고받는 데에 쓰인다. 주로 TCP를 사용하고 HTTP/3 부터는 UDP를 사용하며, 80번 포트를 사용한다.  [구글 정의]



HTTP 특징

* 클라이언트 서버 구조
* 무상태 프로토콜(Stateless), 비연결성
* HTTP 메시지
* 단순함, 확장 가능
